Envy And Spite

A public level loader/editor

Envy & Spite is a project about public level creation for ULTRAKILL. Our goal is to give everyone tools and knowledge on how to create their own ULTRAKILL custom level. For that, we have created Envy (custom level loader) and Spite (custom level editor). All our creations are fully open-source. We hope that you will enjoy your stay here!

Envy Level Loader

Envy is a level loader for the .envy level files generated by Spite Level Editor. Currently, you need to join our Discord server in order to be able to install the level made by other people. However, we are working on an ability to download levels online from a server into the game without any involvement from the user. It is unknown when we will release that feature...

Spite Level Editor

Spite is the tool that is used in order to create said levels. Spite is based on Unity Engine and requires some knowledge in order to be able to work. You can learn how to use Unity, Spite and create ULTRAKILL levels in our official Spite LE Wiki (which is still in development...).
